## Stock Market Sentiment Analysis & Trend Prediction

This project analyzes the relationship between financial news sentiment and stock price movements. 
It combines NLP-based sentiment analysis with historical stock data to explore whether news sentiment 
can help predict short-term market trends.

### Key Features
- Stock price data via Yahoo Finance
- News sentiment analysis using NLP
- Feature engineering and trend prediction
- Visual analysis of sentiment vs price

### Disclaimer
This project is for educational purposes only and does not provide investment advice.
